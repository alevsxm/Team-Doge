Welcome

USERS
‘/users/show/:id’
   * Basic user information
   * Link to view editable Resovision (‘/resovisions/edit/:id’)
   * Button to edit their personal info
   * Button to delete their account

RESOVISIONS
‘/resovisions/index’
   * List of all resovisions (public)

   * If resovision is yours, show: 
      * Links to edit/update sections
      * Link to delete sections
      * Link to delete resovision (empty sections)
